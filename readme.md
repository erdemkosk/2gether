> A p2p zoom clone using peer js. Screen sharing, video, audio and chat can be used.

### ✨ [Demo](https://two-gether.herokuapp.com)

## Prerequisites

- node 14.15.0
- npm6.14.8

## Install

```sh
npm install
```

## Docker

```sh
docker build -t 2gether .
docker run --env SWAGGER_HOST=localhost -p 3030:3030 2gether
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
